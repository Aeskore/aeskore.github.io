L'histoire de Minecraft

Le projet Minecraft est crée le 10 mai 2009 par Markus Persson (Notch) et Jens Bergensten (Jeb_). Le jeu est sorti le 18 novembre 2011 pour les navigateurs web


Markus Persson à ensuite crée son propre studio de jeux video : Mojang studio Depuis Minecraft est devenu un succès et il ont développer le jeu pour Windows, Mac, Linux grâce à java


Minecraft est devenu le jeu le plus vendu du monde avec plus de 10M d'exemplaire. Le jeu est toujours mise à jour depuis 10 ans. Pour l'acheter cliquer ici
